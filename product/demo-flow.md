# Demo Flow (3 minutes, judge-proof)

## Setup (before judges arrive)
- Open the iOS project and run on a simulator.
- Ensure there is at least 1–3 tasks available (or tap “Load Demo Tasks”).
- (Optional) Clear/rotate telemetry logs for a clean run.

## Live demo script (happy path)
1. **Intent Setup (10–20s)**
   - Confirm the 3 projects: Hackathon Repo / Startup Repo / Job Search.
   - Set today’s intent: “Ship the AgentBeats demo: dashboard rings + AI reallocation + telemetry.”
2. **Timer Arena (15–25s)**
   - Select **Hackathon Repo** as the active project.
   - Start a short task timer; show ring progress and remaining time.
   - Pause once, then resume.
3. **Dashboard (30–45s)**
   - Show 3 project rings with weekly caps + completion score.
   - Point to the AI card: “Hackathon is behind plan → reallocate 2h.”
   - Tap **Apply** and call out that the rings update immediately.
4. **Reflection (10–20s)**
   - Mark outcomes (Done/Partial/Missed) and show reflection score.
5. **Telemetry (10–20s)**
   - Show the event sequence and one metric (completion score or reflection score).

## What to say (1 sentence per step)
- Intent: “We start with what matters today.”
- Suggest: “The agent picks the next best task and explains why.”
- Timer: “One tap to start focusing.”
- Ring: “Progress is visible and motivating.”
- Reflection: “We close the loop and learn from outcomes.”
- Telemetry: “We can prove what happened, not just claim it.”
