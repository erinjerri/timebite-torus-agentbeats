# Demo Scenario — “3 Projects, One Agent”

## Characters
**User:** a builder with competing priorities.

## Projects (active)
1. **Hackathon Repo** (highest priority this week)
2. **Startup Repo**
3. **Job Search**

## Initial weekly caps (example)
- Hackathon: 10h
- Startup: 12h
- Job Search: 6h

## Story beats (what happens in the demo)

### 1) Intent Setup
User sets today’s intent:
- “Ship the AgentBeats demo: dashboard rings + AI reallocation + telemetry.”

User confirms weekly caps and active projects.

### 2) Timer Arena
User starts a timer on **Hackathon Repo**.
Ring shows the timer progress.

### 3) Dashboard (the “wow” moment)
Dashboard shows:
- 3 project rings (spent vs cap)
- Completion score
- AI suggestion card:
  - “Hackathon is behind plan. Reallocate **2 hours** from Job Search to Hackathon.”

User taps **Apply**.

### 4) Live update
Immediately:
- Hackathon cap increases by 2h (or its plan allocation increases).
- Job Search cap decreases by 2h.
- Rings and score update live.

### 5) Reflection
User marks:
- Hackathon: Done
- Startup: Partial
- Job Search: Missed

Reflection score displays and is logged.

## Expected judge takeaway
TimeBite doesn’t just track tasks—it reallocates time intelligently and proves outcomes with a measurable ring-based UI.
