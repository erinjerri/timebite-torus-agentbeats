# TimeBite — User Stories (Demo Scope)

## Primary persona
**Busy builder**: wants to pick the next task fast, focus for 15–60 minutes, and get measurable progress.

## Stories (core demo)

### Daily Intent
- **US-INTENT-01**: As a user, I can set 2–3 Daily Intent outcomes for today.
- **US-INTENT-02**: As a user, I can edit today’s intent without losing my tasks.

### Agent suggestion
- **US-AGENT-01**: As a user, I can request a suggested next task.
- **US-AGENT-02**: As a user, I can see *why* the suggestion was made (1-line reason).
- **US-AGENT-03**: As a user, I can accept the suggestion and start the timer in one tap.

### Timer + ring UI
- **US-TIMER-01**: As a user, I can start a timer for a task.
- **US-TIMER-02**: As a user, I can pause and resume without losing elapsed time.
- **US-RING-01**: As a user, I can see ring progress and remaining time at a glance.

### Reflection scoring
- **US-REFLECT-01**: As a user, I can submit a reflection (done/partial/missed) quickly.
- **US-REFLECT-02**: As a user, I can see a reflection score and short feedback.

### Telemetry
- **US-TELEM-01**: As a builder, I can inspect a JSONL file of key events after a run.
- **US-TELEM-02**: As a builder, I can compute basic metrics from those events (completion %, reflection score).
